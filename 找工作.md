## 五险一金

五险包含：

| 名目       | 个人比例 | 企业比例 | 作用       | 年限      | 说明                 |
| ---------- | -------- | -------- | ---------- | --------- | -------------------- |
| 养老       | 8        | 16       | 领取养老金 | 15年      | 死得早可以家里领出来 |
| 医疗       | 2        | 8        | 看病       | 20年~25年 | 年限够之后享受医保   |
| 失业       | 0.5      | 0.5      | 领取失业金 |           |                      |
| 工伤       | 0        | 0.2~14   | 赔付工伤   |           |                      |
| 生育       | 0        | 5~12     |            |           |                      |
| 住房公积金 | 5~12     | \        |            |           |                      |

社保账户分为个人账户和统筹账户。





申请失业保险，在领取失业金的期间，你的医保由当地的社保局或人社局帮你缴纳，所以申领失业金之后不用担心医保会断。
