# Linux

## 查看系统和硬件

### neofetch

Ubuntu用户可以安装neofetch来查看。

```shell
sudo apt update
sudo apt upgrade
sudo apt install neofetch
```



#### 参数速览表

| 参数类别       | 参数选项                             | 说明                                              |
| :------------- | :----------------------------------- | :------------------------------------------------ |
| **禁用信息**   | `--disable cpu memory`               | 不显示 CPU 和内存信息                             |
| **文字格式**   | `--colors 1 2 3 4 5 6`               | 按顺序设置标题、@、下划线、字幕、冒号、信息的颜色 |
|                | `--underline on/off`                 | 启用或禁用标题下划线                              |
|                | `--underline_char "*"`               | 设置标题下划线使用的字符                          |
|                | `--bold on/off`                      | 启用或禁用粗体文本                                |
| **色块设置**   | `--color_blocks on/off`              | 启用或禁用颜色块                                  |
|                | `--block_width 2` `--block_height 1` | 设置颜色块的宽度和高度                            |
| **屏幕截图**   | `--scrot` 或 `-s`                    | 截取屏幕截图                                      |
| **LOGO/ASCII** | `--ascii`                            | 使用 ASCII 艺术字替代发行版 Logo                  |
|                | `--off`                              | 不显示任何 Logo 及其他图形元素，只显示系统信息    |

💡 **关于配置文件的补充**
Neofetch 支持通过配置文件进行持久化设置。配置文件通常位于 `~/.config/neofetch/config.conf`。你可以在这个文件中设置各种选项，一劳永逸，无需每次输入命令行参数。

✨ **实用命令示例**
1.  **只想查看操作系统、内核和硬盘使用情况**：
    ```bash
    neofetch --disable cpu gpu memory resolution uptime packages shell de wm wm_theme theme icons terminal
    ```
    这会禁用大量信息，只保留最基础的几项。

2.  **使用 ASCII 艺术字并启用颜色块**：
    ```bash
    neofetch --ascii --color_blocks on
    ```

3.  **快速获取纯文本系统信息（适用于脚本或记录）**：
    ```bash
    neofetch --stdout
    # 或者使用更简洁的模式
    neofetch --off
    ```

4.  **指定要显示的特定信息项（更现代的方法）**：
    较新版本的 Neofetch 允许你选择只显示哪些信息，而不是禁用哪些。
    ```bash
    neofetch --cpu --gpu --memory --disk
    ```

🧐 **获取更多帮助**
Neofetch 的参数非常多。要查看所有可用的参数和完整的帮助信息，你可以在终端中输入：
```bash
neofetch --help
```
**注意**：输出内容会很长，你可以使用管道符 `|` 配合 `less` 查看器来滚动浏览，例如 `neofetch --help | less`。

希望这些信息能帮助你更好地使用 Neofetch 来展示你的系统！
