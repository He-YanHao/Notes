# 块设备使用流程

1. 识别到块设备
2. 识别到分区
3. 识别到文件系统
4. 自动或手动挂载
5. 应用层可以读写



## 列出块设备lsblk

列出块设备，以树形结构清晰显示所有磁盘和分区。

| 选项 | 说明                       |      |
| ---- | -------------------------- | ---- |
| `-a` | 显示所有设备（包括空设备） |      |
| `-f` | 查看文件系统信息           |      |
| `-l` | 使用列表格式输出（非树状） |      |
| `-t` | 显示设备的拓扑信息         |      |
