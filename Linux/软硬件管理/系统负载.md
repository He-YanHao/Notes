# 查看系统和硬件

## 查看系统

### neofetch

Ubuntu用户可以安装neofetch来查看。

```shell
sudo apt update
sudo apt upgrade
sudo apt install neofetch
```

使用：

```shell
neofetch
# 查看系统及硬件信息
neofetch --off
# 输出结果不包含Logo
```
