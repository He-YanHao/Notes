## 汉化

首先编辑 `/etc/locale.gen` 文件，去掉对应语言的注释。



再使用`  adobe-source-han-sans-cn-fonts`，安装字体，就完成了汉化。可肯能有较大的延迟，会出现豆腐块。

## 输入法

编辑`/etc/environment`这个文件，添加以下几行：

```shell
EDITOR=vim

# GTK_IM_MODULE=fcitx
# QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx
SDL_IM_MODULE=fcitx
GLFW_IM_MODULE=fcitx
```

输入这些回车，安装fcitx5

```shell
sudo pacman -S fcitx5 fcitx5-configtool fcitx5-qt fcitx5-gtk fcitx5-chinese-addons fcitx5-material-color kcm-fcitx5 fcitx5-lua
```

下面是分行版本的：

```shell
sudo pacman -S fcitx5
sudo pacman -S fcitx5-configtool
sudo pacman -S fcitx5-qt
sudo pacman -S fcitx5-gtk
sudo pacman -S fcitx5-chinese-addons
sudo pacman -S fcitx5-material-color
sudo pacman -S kcm-fcitx5
sudo pacman -S fcitx5-lua
```
