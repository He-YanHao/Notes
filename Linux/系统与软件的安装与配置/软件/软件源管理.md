# 软件源管理

## Debian系Linux



```shell
# 创建用于存储APT包管理器信任的软件源的GPG密钥的目录。
sudo mkdir -p /etc/apt/keyrings

# 下载typora的GPG密钥，并将其/etc/apt/keyrings/typora.gpg里，并抛弃终端上的输出。
curl -fsSL https://typoraio.cn/linux/typora.gpg | sudo tee /etc/apt/keyrings/typora.gpg > /dev/null

# 将APT软件源条目字符串追加到/etc/apt/sources.list.d/typora.list下
# 在/etc/apt/sources.list.d/目录下为不同的软件建立源文件是一个好习惯
echo "deb [signed-by=/etc/apt/keyrings/typora.gpg] https://typoraio.cn/linux ./" | sudo tee /etc/apt/sources.list.d/typora.list

# 可以使用了
sudo apt update
sudo apt install typora
```
