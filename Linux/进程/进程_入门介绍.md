# Linux进程入门介绍

## 基础介绍

## ps指令

可以通过`ps`指令查看进程。

参数：

```shell
ps -x
# 显示当前用户的所有进程
ps -ax
# 显示当前系统的所有进程 包含所有用户的进程
pa -u
# 更详细的进程描述
pa -w
# 显示命令的全名
```

输出：

```shell
PID    TTY          STAT     TIME         COMMAND
进程ID  进程所在终端   进程状态  占用CPU总时长  命令名
```
