# Linux常用指令和操作

## 关机指令 重启指令

```shell
shutdown
# 通知各个进程关闭系统 等待关闭完毕后关闭电源 
poweroff
# 强行结束系统 关闭电源
halt
# 关闭系统 不关闭电源
reboot
# 重启
```

## 有关命令的命令

| 命令  | 格式                                   | 作用                           |
| ----- | -------------------------------------- | ------------------------------ |
| type  | type 命令                              | 输出 命令 的来源或类型         |
| which | which 命令                             | 输出 可执行文件命令 的具体位置 |
| help  | **help 命令** 或 **命令 --help**       | 输出 命令 的使用方式教程       |
| man   | man 命令                               | 输出 命令 的使用手册           |
| alias | alias 命令别名='命令1;命令2;命令3;...' | 给命令一个别名                 |
