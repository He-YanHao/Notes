# 嵌入式C

## C语言规范和习惯

- TAB键缩进不同软件规范不一样，所以可以统一用空格代替，四个或八个空格代替一个TAB。

  可以在设置里设定为缩进替换为四个或八个空格字符。

- 指针*放在靠近变量名的一侧。

- 关键词后面加空格。

- 常量宏和枚举统一使用大写。

- 一般函数命名均为小写，专有名词例外，如：OLED LCD。

文件信息注释

```c
/**
  ****************************************************************************************************
  *@标题    
  *@版本    
  *@时间    
  ****************************************************************************************************
  *@说明
  *
  *
  ****************************************************************************************************
  */

```

函数注释

```c
/**
  *@函数作用    
  *@参数1    
  *@参数2    
  *@返回值    
  */
```

注意：

- 写清每一个变量的作用，注意取值范围，避免下面的错误情况。

  ```c
  for(uint8_t i = 0, i < 1000, i++)
  {
  	循环语句；
  }
  ```
