# 进制

常用进制一般为二进制，八进制，十进制，十六进制。

其中二进制以 ` 0b ` 开头，八进制以 ` 0 ` 开头，十六进制以 ` 0x ` 开头。                                     

| 二进制 | 十六进制 | 十进制 |
| ------ | -------- | ------ |
| 0000   | 0        | 0      |
| 0001   | 1        | 1      |
| 0010   | 2        | 2      |
| 0011   | 3        | 3      |
| 0100   | 4        | 4      |
| 0101   | 5        | 5      |
| 0110   | 6        | 6      |
| 0111   | 7        | 7      |
| 1000   | 8        | 8      |
| 1001   | 9        | 9      |
| 1010   | A        | 10     |
| 1011   | B        | 11     |
| 1100   | C        | 12     |
| 1101   | D        | 13     |
| 1110   | E        | 14     |
| 1111   | F        | 15     |
