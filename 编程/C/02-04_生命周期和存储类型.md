# 生命周期

变量存在生命周期，一旦生命周期到期内存就会被释放，不复存在。

按从长到短排列是：

- 在函数外申请的数据不会被释放。

- 在函数中申请的数据在函数结束后空间被释放。
- 在for循环中申请的数据在for循环结束后空间被释放。

# 储存类型

包含 **auto** 存储类  **register** 存储类  **static** 存储类  **extern** 存储类。

- **auto**

	auto 存储类是所有局部变量默认的存储类。

	定义在函数中的变量默认为 auto 存储类，这意味着它们在函数开始时被创建，在函数结束时被销毁。

- **register**

	使用 register 修饰的变量被存储在寄存器里，而不是 RAM。因此有着更高的速度，不能直接取地址，所以不能使用取地址符&，也无法使用一元的 '&' 运算符。其大小不可超过寄存器的大小，一般是一个字 32 位。

- **static**

	使用 static 修饰的变量在生命周期结束后不会被销毁。

- **extern**

	使用 extern 修饰的变量在被创建时不会被分配内存，而是提示编译器去其他相关联的文件中寻找该变量。
