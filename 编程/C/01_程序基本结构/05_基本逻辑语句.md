# 逻辑处理

C语言用于逻辑判断的关键字有

-   if
-   for
-   while
-   switch



## if

```c
if(逻辑判断)
{
	// 如果成立执行这一段
} else
{
	// 不然执行这一段
}
```



## for

```c
for(初始化;逻辑表达式;更改条件)
{
	// 初始化后判断逻辑表达式，成立则执行循环体，执行一次循环后执行更改条件，再次循环，直到逻辑表达式判断不成立则不再循环。
}
```



## while

```c
while(逻辑表达式)
{
	// 如果成立执行这一段循环
}
```

```c
do{
    // 先执行执行这一段循环，再判断逻辑表达式。
}while(逻辑表达式)
```



## switch

```c
switch(整型变量)
{
    case 常量1: 语句1; break;
	case 常量2: 语句2; break;
    ......
    default:语句体x;break;
}
```
