# Java

在终端里，使用 ` java 文件名` 执行Java程序，使用 ` javac 文件名` 编译Java程序。

## 注释

Java注释和C语言一样包含 ` // ` 和 ` /*   */ ` ，同时还包含 ` /**    */ ` 它可以被javadoc工具软件来生成信息，并输出到 HTML 文件中。

## Java结构

* 项目 --- 
* 模块 --- 
* 包 --- 
* 类 --- 相当于C语言中的头文件
* 对象 --- 相当于C语言中的函数

**源文件名**：源文件名必须和类名相同。当保存文件的时候，你应该使用类名作为文件名保存（切记 Java 是大小写敏感的），文件名的后缀为 **.java**。（如果文件名和类名不相同则会导致编译错误）。
