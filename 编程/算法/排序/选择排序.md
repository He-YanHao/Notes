# 选择排序

1. 将数组分为已排序区间和未排序区间
2. 初始时已排序区间为空
3. 每次从未排序区间中找出最小元素
4. 将该最小元素与未排序区间的第一个元素交换位置
5. 重复以上过程直到所有元素都排序完成

案例：

```c
// 选择排序函数
void selection_sort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        // 在未排序部分中查找最小元素的索引
        int min_index = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[min_index]) {
                min_index = j;
            }
        }
        // 将找到的最小元素与当前位置交换
        if (min_index != i) {
            int temp = arr[i];
            arr[i] = arr[min_index];
            arr[min_index] = temp;
        }
    }
}
```

void huan(int *a, int *b)
{
    int temp = *a;
    *a = *b;
    *b = temp;
}
void paixv(int *num, int quantity)
{
    int ok = 0;
    for(int j = 0; j < quantity; j++)//循环找quantity次
    {
        int min = ok;
        for(int i = ok; i < quantity; i++)//剩下的数字里面找到最小
        {
            if(num[min] >= num[i])
            {
                min = i;
            }
        }
        huan(&num[min],&num[ok]);//交换最小和已排序
        ok++;
    }
}
