# 冒泡/交换排序 (Bubble Sort)

1. 比较相邻元素，如果第一个比第二个大，就交换它们
2. 对每一对相邻元素做同样工作，从开始到结束
3. 每次遍历后，最大的元素会"冒泡"到末尾
4. 重复上述步骤，每次遍历减少一个元素（已排序部分）

```c
void bubble_sort(int arr[], int n)//冒泡排序
{
    int flag = n;//未排序部分
    for(int j = 0; j < n; j++)
    {
        for(int i = 0; i < flag-1; i++)//每次只排序未排序部分
        {
            if(arr[i] > arr[i+1])//如果左元素大于右元素则交换
            {
                int num = arr[i];
                arr[i] = arr[i+1];
                arr[i+1] = num;
            }
        }
        flag--;//未排序部分-1
    }
}
```
