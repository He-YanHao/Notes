### 🚀 高效学习技巧
1. **对照C知识学习**  
   - 每学一个C++特性（如类），思考：“如果用C实现这个功能要怎么写？”
   - 例：用`std::vector` vs C动态数组，体会RAII的便利

2. **动手 > 阅读**  
   ```cpp
   // 示例：体验类与构造函数的优势
   class Rectangle {
   public:
       Rectangle(int w, int h) : width(w), height(h) {} // 初始化列表
       int area() const { return width * height; } // const成员函数
   private:
       int width, height;
   };
   
   // 对比C版本
   struct RectangleC { int width, height; };
   int area_c(const RectangleC* rect) { ... }
   ```

3. **重点标记现代C++**  
   书中C++11新特性有特殊标识（如图标），**优先掌握这些**：
   - `auto` 类型推导（Ch 3）
   - 范围`for`循环（Ch 5）
   - 智能指针（Ch 16）
   - Lambda表达式（Ch 18）

4. **善用课后习题**  
   只做**标志性习题**（每个知识点选1~2题），例如：
   - 实现一个带构造/析构的类
   - 用`vector`替换数组
   - 使用`std::sort` + lambda

### 🔧 克服“大部头”恐惧的实操建议
1. **两本书并行**  
   - 主教材：《C++ Primer Plus》按上述章节精读
   - 辅助：《A Tour of C++》（C++之父写的精简指南，200页）
   - 当某个概念在Primer中太冗长时，看Tour中的简洁解释

2. **建立知识地图**  
   ```mermaid
   graph LR
   A[C基础] --> B[C++核心]
   B --> C[类/对象]
   B --> D[引用/const]
   B --> E[函数重载]
   C --> F[构造函数]
   C --> G[继承]
   C --> H[多态]
   F --> I[智能指针]
   G --> J[虚函数]
   ```

3. **快速构建正反馈**  
   学完每个核心章节后，**立刻写个小项目**：
   - 学完类 → 写个`Student`类管理成绩
   - 学完STL → 用`vector`+`sort`实现排行榜
   - 学完继承 → 实现`Shape`->`Circle`/`Rectangle`
