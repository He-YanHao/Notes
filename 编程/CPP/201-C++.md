# C++

## C++格式转化

使用 ` static_cast ` 转化数据类型。

```
int i = 10;
float f = static_cast<float>(i);
```

可以将静态int类型转换为float类型

## C++字符串`std::string`

C++字符串需要`#include<string>`头文件


```
std::string 变量名称{"字符串",开始位置,截取长度};
```

例如

```
std::string a{"123456",2,2};
```

则为34，字符串位置从0开始。

中括号为指定位置，如

```
std::string a{123456};
std::cout << a[2];//输出3
```

string可以使用+=进行附加，如

```
str1 += str2;
```

直接将str2附加到了str1后面，且str2不变。

string可以判断是否相等

### 以文本文件形式输出

需要包含`#include <fstream>`头文件

首先要用`std::ofstream`申请一个ofstream对象，并用`ofstream对象名.open("文本文档名.txt");`将ofstream对象名和文本文档关联起来，就可以像`std::cout`一样使用`ofstream对象名`将内容输出到文本文件中。

## new和delete

若没有给指针赋于变量的地址值则无法直接操作指针，需要使用`int *pa =new int;`的方式申请一个指针并分配给他空间，使用数据类型的不同来确定空间的大小。

若要释放掉这部分内存需要`delete 内存名`

若申请的是一个数组，则`int *pa =new int[数字];`的形式申请，释放则需要`delete [] 内存名`的方式释放数组的全部内存。
