# BCD 码

BCD 码（Binary-Coded Decimal，二进制编码的十进制）是一种将每一位十进制数字分别用固定长度的二进制编码表示的方式。

它的核心特征只有一句话：

一位十进制数，用一组二进制位来表示，而不是把整个数直接转成二进制。



## 最常见形式：8421 BCD

最常用的是 **8421 BCD**，即用 4 个二进制位表示一位十进制数：

| 十进制 | BCD（8421） |
| ------ | ----------- |
| 0      | 0000        |
| 1      | 0001        |
| 2      | 0010        |
| 3      | 0011        |
| 4      | 0100        |
| 5      | 0101        |
| 6      | 0110        |
| 7      | 0111        |
| 8      | 1000        |
| 9      | 1001        |

👉 **1010 ~ 1111 在 8421 BCD 中是非法码**





## BCD 码的本质属性

从学科角度看：

-   **BCD 不是算法**
-   **BCD 也不是数据结构**
-   **BCD 是一种数据编码 / 数据表示方式**

更准确的分类是：

数值编码规则（Data Encoding Format）

------

## BCD 的常见变种

### 8421 BCD（最常见）

-   权值：8-4-2-1
-   通用、直观

------

### 余 3 码（Excess-3）

| 十进制 | Excess-3 |
| ------ | -------- |
| 0      | 0011     |
| 1      | 0100     |
| 2      | 0101     |
| …      | …        |

-   每个数 = BCD + 3
-   具备自补性质
-   用于某些硬件逻辑设计

------

### 压缩 BCD（Packed BCD）

-   一个字节存 **两个十进制数**
-   高 4 位 + 低 4 位

例如：

```
0x59 → 表示十进制 59（不是 0x59 = 89）
```
