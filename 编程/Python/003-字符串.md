# 字符串

## 字符串基本

python没有单字符，单字符被视为仅有一个字母的字符串。

Python 中的字符串不能改变。

反斜杠可以用来转义，使用 **r** 可以让反斜杠不发生转义。 如 **r"this is a line with \n"** 则 **\n** 会显示，并不是换行。

### 引号部分

可以用 ` '` 或 ` " ` 指定一个字符串。

 ` '` 开始的字符串要用用 ` '` 结束，也就是意味着 ` '` 包含的字符串可以包含 ` " ` 。 ` " ` 同理。

可以用 ` '''` 或 ` """ ` 指定一个多行字符串。

## 字符串用法

### 基本

Python中的字符串可以直接相加

```
a = 'I'
b = ' love'
c = ' Python'
print(a + b + c)
# result I love Python 
```

Python的字符串不仅可以相加，也可以乘以一个数字：

```
a = 'word'
print(a*3)
#result wordwordword
```

### 切片和索引

字符串的切片和索引就是通过string[x]

获取字符串的一部分信息：

```
a = 'I love python'
print(a[0])//取出第一个字符
#result I
print(a[0:5])//取出第一到第五个字符
#result I lov
print(a[-1])//取出最后一个字符
#result n
```

### 大小写

在字符串的末尾使用 ` title() ` 方法可以将字符串首字母大写，如：

```
name = "he"
print(name.title())
```

输出 ` He ` 。

于此类似的还有全小写 ` lower() ` 和全大写 ` unper() ` 。

### 消除字符串开始和结束的空格

使用` lstrip() ` 消除字符串开始的空格。

使用` rstrip() ` 消除字符串开始和结束的空格。

使用` strip() ` 消除字符串开始和结束的空格。

但仅仅是暂时删去，需要将删去空白的字符串重新赋给原字符串。

## 字符串方法

### split()

```
a ='www.baidu.com'
print(a.split('.'))
#result ['www','baidu','com']
```

### str()

可以将非字符串转化为字符串，如：

```
num = 10
print(num + num)
#输出 20
print(str(num) + str(num))
#输出 1010
```
