# `__name__  与 __main__ `

## 核心结论

```python
if __name__ == "__main__":
```

每个 Python 文件都有一个内置变量：

```python
__name__
```

这个内置变量存在两种可能性：

| 场景     | `__name__` 值                    |
| -------- | -------------------------------- |
| 直接运行 | `"__main__"`                     |
| 被导入   | 模块名（如 `app.layout.layout`） |

所有可以使用这个 `if` 判断程序的运行情况，用来判断程序的启动情况。

同时，Python没有强制 main 函数，是从头到尾指向一遍，因此可以通过 `if __name__ == "__main__"` 实现“入口函数约定”。
