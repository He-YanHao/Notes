# 负反馈放大电路

## 什么是负反馈放大电路

**负反馈放大电路**是一种将**输出信号的一部分或全部反相后送回输入端**，与原始输入信号相减的放大电路。这种"自我调节"机制是现代电子系统中最重要的技术之一，它能显著改善放大电路的各项性能指标，提高系统的稳定性和可靠性。



## 负反馈的基本概念

### 反馈定义
反馈就是将放大电路输出量（电压或电流）的一部分或全部，通过反馈网络送回到输入端的过程。

### 负反馈原理
```
输入信号 → [比较环节] → [基本放大器] → 输出信号
              ↑               |
              |               |
              +---[反馈网络]---+
```
**数学表达**：
- 开环增益：A = Xo/Xi
- 反馈系数：F = Xf/Xo  
- 闭环增益：Af = Xo/Xid = A/(1+AF)

当**AF >> 1**时，Af ≈ 1/F，闭环增益仅取决于反馈网络，与基本放大器无关。



## 负反馈放大电路的组成

### 基本组成部分
```
Xid = Xi - Xf
     ↓
  [基本放大器] A
     ↓
      Xo
     ↙   ↘
   负载   [反馈网络] F
     ↘   ↙
      Xf
```

### 各部件功能
- **基本放大器**：提供主要放大功能，增益为A
- **反馈网络**：采样输出信号并送回输入端，反馈系数为F
- **比较环节**：将输入信号与反馈信号进行比较，产生净输入信号



## 关键参数

#### 环路增益（AF）

环路中各个增益的乘积，反映反馈的强度。
\[ AF = A \times F \]

**物理意义**：信号沿环路一周的总增益。

#### 反馈深度（1+AF）

衡量负反馈强弱的指标。
- **1+AF = 1**：无反馈
- **1+AF > 1**：负反馈
- **1+AF >> 1**：深度负反馈

#### 灵敏度（S）

闭环增益相对于开环增益变化的敏感程度。
\[ S = \frac{dA_f/A_f}{dA/A} = \frac{1}{1+AF} \]

深度负反馈时，S ≈ 0，系统对参数变化不敏感。



## 负反馈的四种基本组态

### 电压串联负反馈

**特点**：
- 采样：电压
- 叠加：串联
- 稳定输出电压
- 提高输入阻抗，降低输出阻抗

**典型电路**：
```
        +--[Rf]--+
        |        |
Vin --[R1]--|-\   |
            |  >--+-- Vout
     GND ---|+/
```

**增益表达式**：
\[ A_{vf} = \frac{A_v}{1+A_vF_v} \]

### 电压并联负反馈

**特点**：
- 采样：电压
- 叠加：并联
- 稳定输出电压
- 降低输入输出阻抗

**典型电路**：
```
Vin --[R1]--+--|-\
            |  |  >-- Vout
           [Rf] |  |
            |  |+/
           GND GND
```

**增益表达式**：
\[ A_{rf} = \frac{A_r}{1+A_rF_g} \]

### 电流串联负反馈

**特点**：
- 采样：电流
- 叠加：串联
- 稳定输出电流
- 提高输入输出阻抗

**典型电路**：
```
Vin ----|+\
        |  >--+--[R]-- Vout
GND --|-/     |
          [Rf]
             |
            GND
```

**增益表达式**：
\[ A_{gf} = \frac{A_g}{1+A_gF_r} \]

### 电流并联负反馈

**特点**：
- 采样：电流
- 叠加：并联
- 稳定输出电流
- 降低输入阻抗，提高输出阻抗

**典型电路**：
```
        +--[Rf]--+
        |        |
Vin --[R1]--+--|-\
            |  |  >--+--[R]-- Vout
           GND |+/   |
                 [Re]
                  |
                 GND
```



## 负反馈对放大电路性能的影响

### 提高增益稳定性

**改善程度**：
\[ \frac{dA_f}{A_f} = \frac{1}{1+AF} \cdot \frac{dA}{A} \]

**实例**：
某放大器A=1000，F=0.01，当A变化±10%时：
- 1+AF = 1+1000×0.01 = 11
- Af变化仅为：±10%/11 ≈ ±0.9%

### 扩展通频带

**频带扩展原理**：
负反馈能自动补偿高频和低频增益的下降。

**扩展倍数**：
- 上限频率扩展：(1+AF)倍
- 下限频率降低：1/(1+AF)倍

**频率响应对比**：
```
  Gain
   ↑
   │  无反馈特性
   │  ┌─────────────┐
   │  │             │
   │  │             │
   │  │             │
   │  │ 负反馈特性   │
   │  └─────────────┘
   └──────────────────→ Frequency
     fL      fH
```

### 减小非线性失真

**工作原理**：
负反馈能自动校正放大器产生的非线性失真。

**失真改善**：
\[ D_f = \frac{D}{1+AF} \]

其中D为无反馈时的失真度。

### 抑制噪声和干扰

**噪声改善**：
- 对内部噪声：抑制(1+AF)倍
- 对外部干扰：如干扰与信号同时加入，无法抑制

### 改变输入输出阻抗

**输入阻抗**：
- 串联负反馈：增大(1+AF)倍
- 并联负反馈：减小1/(1+AF)倍

**输出阻抗**：
- 电压负反馈：减小1/(1+AF)倍
- 电流负反馈：增大(1+AF)倍



## 深度负反馈分析

### 深度负反馈条件
当**1+AF >> 1**，通常AF > 10即可认为深度负反馈。

### 虚短和虚断概念
- **虚短**：V+ ≈ V-（运放输入端电压近似相等）
- **虚断**：I+ ≈ I- ≈ 0（运放输入端电流近似为零）

### 近似计算方法
在深度负反馈条件下：
\[ A_f \approx \frac{1}{F} \]

**计算步骤**：
1. 识别反馈组态
2. 利用虚短虚断概念
3. 列写电路方程
4. 求解闭环增益



## 负反馈放大电路的稳定性

### 自激振荡产生原因
当环路增益满足：
- **幅度条件**：|AF| ≥ 1
- **相位条件**：∠AF = ±180°

### 稳定性判据
- **增益裕度**：GM > 10dB
- **相位裕度**：PM > 45°

### 频率补偿技术

#### 电容补偿
**主极点补偿**：
在放大器内部加入小电容，降低第一个极点频率。

**米勒补偿**：
利用米勒效应，用小电容实现大电容的补偿效果。

#### 零极点补偿
通过加入零点来抵消原有的极点，改善相位裕度。

#### 前馈补偿
引入前馈通路，提高高频性能。



## 典型负反馈应用电路

### 同相比例放大器
```
Vin ----|+\
        |  >--+-- Vout
     ---|-/   |
    |      [R1]
    |         |
    +---[Rf]--+
```

**增益**：
\[ A_f = 1 + \frac{R_f}{R_1} \]

### 反相比例放大器
```
Vin --[R1]--+--|-\
            |  |  >-- Vout
           [Rf] |  |
            |  |+/
           GND GND
```

**增益**：
\[ A_f = -\frac{R_f}{R_1} \]

### 电压跟随器
```
Vin ----|+\
        |  >-- Vout
     ---|-/
```

**增益**：Af ≈ 1

### 仪表放大器
```
        Rg
   +----/\/\/----+
   |             |
Vi1--[R]--[A1]   |   [A3]-- Vo
        |        |    |
        +--------+----+
        |        |
Vi2--[R]--[A2]   |
        |        |
        +--------+
           [R]
```

**增益**：
\[ A_f = 1 + \frac{2R}{R_g} \]



## 设计规则与最佳实践

**反馈深度选择**
*   一般应用：1+AF = 10～100
*   高精度应用：1+AF > 1000
*   高速应用：适当减小反馈深度以保证带宽

**稳定性设计**
*   必须保证足够的相位裕度（>45°）
*   高频应用需要进行频率补偿
*   布板时注意减少寄生参数

**反馈网络设计**
*   反馈元件要稳定可靠，温度系数小
*   精密应用要使用高精度电阻
*   高频应用要考虑分布参数影响

**输入输出保护**
*   防止输入过载损坏放大器
*   输出短路保护
*   静电防护措施

**布局布线要点**
*   反馈路径要短而直接
*   敏感信号远离干扰源
*   良好的接地和去耦



## 负反馈电路的调试

### 静态调试
1. 检查各级工作点
2. 测量反馈网络直流状态
3. 确认无自激振荡

### 动态调试
1. 测量闭环增益是否符合设计
2. 检查频率响应特性
3. 测试瞬态响应

### 稳定性测试
1. 方波测试：观察振铃情况
2. 频率扫描：测量相位裕度
3. 温度试验：验证温度稳定性

### 常见故障处理
- **自激振荡**：增加补偿电容，检查布局
- **增益异常**：检查反馈网络，测量工作点
- **失真过大**：检查动态范围，调整反馈深度



## 特殊负反馈技术

### 电流反馈运算放大器（CFA）

**特点**：
- 高速度，宽带宽
- 增益与带宽相对独立
- 适合高速应用

### 自适应反馈

**原理**：
根据输入信号特性自动调整反馈深度。

**应用**：
- 自动增益控制（AGC）
- 动态范围扩展

### 非线性反馈

**应用**：
- 对数放大器
- 压缩扩展器
- 自动电平控制



## 选型与设计指南

### 运放选择
- **通用应用**：通用运放（LM358、TL082）
- **高精度**：精密运放（OP07、AD620）
- **高速**：高速运放（AD811、LM6181）
- **低功耗**：CMOS运放（LMC6482）

### 反馈元件选择
- **电阻**：金属膜电阻，精度根据要求选择
- **电容**：高频用陶瓷电容，精密用聚丙烯电容

### 设计流程
1. 确定系统指标要求
2. 选择适当的反馈组态
3. 计算反馈网络参数
4. 稳定性分析和补偿
5. 仿真验证
6. 实验调试

> **设计提示**：负反馈是模拟电路设计的核心技术。正确应用负反馈可以显著改善电路性能，但不恰当的反馈可能导致振荡或不稳定。设计时要综合考虑增益、带宽、稳定性、噪声等多方面因素，在深度负反馈的益处与稳定性风险之间找到最佳平衡点。对于复杂系统，建议使用电路仿真软件进行前期验证。
