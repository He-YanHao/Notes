# 基础输入控件

## 文本输入

### **QLineEdit**：单行文本编辑器

#### 文本内容操作成员函数

| 函数                            | 说明                         |
| :------------------------------ | :--------------------------- |
| `QString text() const`          | 获取当前文本内容             |
| `void setText(const QString &)` | 设置文本内容                 |
| `void clear()`                  | 清空文本                     |
| `void insert(const QString &)`  | 在光标处插入文本             |
| `QString displayText() const`   | 获取显示文本（考虑密码掩码） |

**示例**：

```cpp
lineEdit->setText("Hello");
QString content = lineEdit->text();  // 获取真实内容
```

------

#### 显示控制成员函数

| 函数                                       | 说明                       |
| :----------------------------------------- | :------------------------- |
| `void setPlaceholderText(const QString &)` | 设置占位提示文本           |
| `void setEchoMode(EchoMode)`               | 设置回显模式（如密码掩码） |
| `EchoMode echoMode() const`                | 获取当前回显模式           |
| `void setAlignment(Qt::Alignment)`         | 设置文本对齐方式           |
| `void setReadOnly(bool)`                   | 设置只读模式               |

**回显模式枚举**：



```cpp
QLineEdit::Normal      // 正常显示（默认）
QLineEdit::NoEcho      // 不显示任何内容（如密码输入）
QLineEdit::Password    // 显示掩码字符（••••）
QLineEdit::PasswordEchoOnEdit // 编辑时显示明文，失去焦点后掩码
```

------

#### 输入控制与验证成员函数

| 函数                                    | 说明                       |
| :-------------------------------------- | :------------------------- |
| `void setMaxLength(int)`                | 设置最大输入长度           |
| `void setValidator(const QValidator *)` | 设置输入验证器             |
| `void setInputMask(const QString &)`    | 设置输入掩码（如日期格式） |
| `void setCompleter(QCompleter *)`       | 设置自动补全               |

**验证器示例**：



```cpp
// 只允许输入数字
QIntValidator *validator = new QIntValidator(0, 100, lineEdit);
lineEdit->setValidator(validator);
```

------

#### 光标与选择成员函数

| 函数                                       | 说明             |
| :----------------------------------------- | :--------------- |
| `void setCursorPosition(int)`              | 设置光标位置     |
| `int cursorPosition() const`               | 获取当前光标位置 |
| `void setSelection(int start, int length)` | 选择文本范围     |
| `QString selectedText() const`             | 获取选中文本     |
| `void deselect()`                          | 取消选择         |

**示例**：



```cpp
lineEdit->setSelection(0, 5);  // 选中前5个字符
```

------

#### 外观与样式成员函数

| 函数                                        | 说明                       |
| :------------------------------------------ | :------------------------- |
| `void setFrame(bool)`                       | 是否显示边框               |
| `void setStyleSheet(const QString &)`       | 设置CSS样式                |
| `void addAction(QAction *, ActionPosition)` | 添加动作按钮（如清除图标） |

**样式表示例**：



```cpp
lineEdit->setStyleSheet("QLineEdit { border: 2px solid gray; }");
```

------

#### 信号成员函数（重要事件通知）

| 信号                           | 触发时机               |
| :----------------------------- | :--------------------- |
| `textChanged(const QString &)` | 文本内容改变时         |
| `textEdited(const QString &)`  | 用户编辑文本时         |
| `returnPressed()`              | 按下回车键时           |
| `editingFinished()`            | 编辑完成（焦点离开）时 |

**信号连接示例**：

```cpp
connect(lineEdit, &QLineEdit::textChanged, [](const QString &text){
    qDebug() << "当前输入：" << text;
});
```

------

#### 高级功能成员函数

| 函数                                           | 说明                            |
| :--------------------------------------------- | :------------------------------ |
| `void setDragEnabled(bool)`                    | 是否允许拖放操作                |
| `void setClearButtonEnabled(bool)`             | 是否显示清除按钮                |
| `void setCursorMoveStyle(Qt::CursorMoveStyle)` | 设置光标移动逻辑（按字符/单词） |

案例：

```cpp
QLineEdit *lineEdit = new QLineEdit(this);
//创建一个单行文本输入框（QLineEdit）
//this：指定父部件（当前窗口/部件），父对象负责管理其内存（自动释放）
lineEdit->setPlaceholderText("请输入用户名");
//设置占位文本（灰色提示文字）
lineEdit->setEchoMode(QLineEdit::Password); // 密码模式
//设置文本回显模式为密码模式
```

### QTextEdit：多行富文本编辑器

```cpp
QTextEdit *textEdit = new QTextEdit(this);
textEdit->setHtml("<b>加粗</b>文本");
```

### QPlainTextEdit：多行纯文本编辑器（性能更优）

```cpp
QPlainTextEdit *plainEdit = new QPlainTextEdit(this);
```

## 数值输入

- **QSpinBox**：整数输入框

  ```cpp
  QSpinBox *spinBox = new QSpinBox(this);
  spinBox->setRange(0, 100); // 设置范围
  ```

- **QDoubleSpinBox**：浮点数输入框

  ```cpp
  QDoubleSpinBox *dSpinBox = new QDoubleSpinBox(this);
  dSpinBox->setDecimals(2); // 小数位数
  ```

- **QSlider**：滑块控件

  ```cpp
  QSlider *slider = new QSlider(Qt::Horizontal, this);
  slider->setTickInterval(10); // 刻度间隔
  ```

## 选择控件

- **QComboBox**：下拉选择框

  ```cpp
  QComboBox *combo = new QComboBox(this);
  combo->addItems({"选项1", "选项2", "选项3"});
  ```

- **QCheckBox**：复选框

  ```cpp
  QCheckBox *checkBox = new QCheckBox("同意条款", this);
  checkBox->setChecked(true);
  ```

- **QRadioButton**：单选按钮

  ```cpp
  QRadioButton *radio1 = new QRadioButton("男", this);
  QRadioButton *radio2 = new QRadioButton("女", this);
  ```
