# 相机

## 基础名词



**焦距（Focal Length）**：决定视野范围，短焦广角，长焦远摄。

**光圈（Aperture）**：控制进光量，影响曝光和景深，表示为 f/2.8、f/1.8 等。

**图像传感器（Sensor）**

-   替代胶片，将光信号转换为电子信号。
-   专业相机传感器大且高性能，像素越大，对光线敏感度越高。

**取景器（Viewfinder）或屏幕**

-   用于构图，单反有光学取景器（OVF），无反相机多为电子取景器（EVF）。

**快门（Shutter）和快门速度（Shutter Speed）**

-   快门控制光线照射传感器的时间，单位是秒或分数秒（如 1/250s）。
-   快门速度越快，动作越冻结；越慢，容易产生拖影效果。

**其他组件**

-   **感光度 ISO**：调节传感器对光的敏感度。
-   **影像处理器（ISP）**：处理拍摄的图像信号，包括降噪、色彩调整等。
-   **存储卡接口**：存储照片和视频。



**焦距（Focal Length）**：镜头中心到传感器的距离，单位 mm

-   例如 35mm 广角，85mm 人像，200mm 远摄。

**景深（Depth of Field, DOF）**：画面中清晰范围

-   大光圈 → 浅景深 → 背景虚化
-   小光圈 → 深景深 → 全景清晰

**白平衡（White Balance, WB）**

-   调整画面色温，使白色真实。
-   常用设置：日光、阴影、钨丝灯、自动 WB。

**对焦模式（AF Mode）**

-   单点对焦（Single AF）：固定焦点
-   连续对焦（Continuous AF）：追踪移动物体
-   自动区域对焦（Auto AF）：相机选择焦点

**RAW 与 JPEG**

-   RAW：未压缩原始图像，后期可调整曝光、白平衡
-   JPEG：压缩图像，方便分享，但可调整空间小



## 基本拍摄参数

拍摄照片时，通常要控制三大参数，也叫“**曝光三要素**”：

1.  **光圈（Aperture）**
    -   控制进光量和景深。
    -   大光圈 f/1.4 → 背景虚化明显；小光圈 f/16 → 景深深，前后都清晰。
2.  **快门速度（Shutter Speed）**
    -   控制曝光时间和动态效果。
    -   快速快门冻结动作，慢速快门可拍出光轨、流水丝滑效果。
3.  **ISO 感光度**
    -   提高 ISO 可在暗光下拍摄，但会增加噪点。
    -   专业相机通常低噪点，高 ISO 可用范围更宽。

>   **曝光平衡公式**：曝光 = 光圈大小 × 快门速度 × ISO
>    这三者配合调整可得到理想曝光。





## 拍照模式/索尼a7m3

-   AUTO：完全自动，无效化所有设置。

-   P：ISO有效，光圈 曝光时间 快门自动。
-   A：光圈优先 小光圈背景清晰，大光圈背景模糊。
-   S：快门优先，高速抓拍，慢速轨迹。
-   M：手动曝光，光圈 曝光时间 快门 ISO均需手动调节。
-   1 / 2 ：可以保存常用的参数。
-   S&Q 快动作和慢动作。
-   AF-ON 对焦，眼控，放大视频，长按锁定人眼
-   AEL 对所需要曝光的位置对焦，锁定曝光
