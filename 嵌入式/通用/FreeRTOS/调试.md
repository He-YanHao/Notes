## 输出任务状态

启用

```c
#define configUSE_TRACE_FACILITY              1
#define configUSE_STATS_FORMATTING_FUNCTIONS  1
```

### 生成所有任务的状态概览 `vTaskList()`

**输出格式示例**：

```
TaskName    State  Priority  Stack   Num
Task1       R      1         100     1
Task2       B      2         90      2
IDLE        R      0         80      3
```
| 名称  | 状态          | 优先级 | 堆栈使用 | 任务编号 |
| ----- | ------------- | ------ | -------- | -------- |
| task1 | X 运行        | 1      | 80       | 1        |
| task2 | Blocked阻塞   | 2      | 44       | 5        |
| IDLE  | Ready就绪     | 0      | 116      | 2        |
| task4 | Suspended暂停 | 4      | 48       | 4        |
| task3 | D 删除        | 3      | 95       | 7        |

### 生成任务CPU占用率统计 `vTaskGetRunTimeStats()`

**输出格式示例**：

```
TaskName    Runtime (%)  
Task1       35.5
Task2       60.2
IDLE        4.3
```

### 案例

```c
void printTaskStats(void)
{
    char pcWriteBuffer[512]; // 足够大的缓冲区
    
    // 打印任务列表
    vTaskList(pcWriteBuffer);
    printf("Task List:\n%s\n", pcWriteBuffer);
    
    // 打印运行时统计
    vTaskGetRunTimeStats(pcWriteBuffer);
    printf("Runtime Stats:\n%s\n", pcWriteBuffer);
}
```

## 查询指定任务的当前状态`eTaskGetState()` 

`eTaskGetState()` 是 FreeRTOS 提供的一个关键 API 函数，用于查询指定任务的当前状态。

通过 `eTaskState eState = eTaskGetState(句柄)` 获取任务状态后打印输出。

| 枚举值       | 数值 | 状态描述                                           |
| :----------- | :--- | :------------------------------------------------- |
| `eRunning`   | 0    | 任务正在运行（当前正在执行的任务）                 |
| `eReady`     | 1    | 任务处于就绪状态（准备运行，等待调度）             |
| `eBlocked`   | 2    | 任务处于阻塞状态（调用了 vTaskDelay() 等阻塞函数） |
| `eSuspended` | 3    | 任务被挂起（调用了 vTaskSuspend()）                |
| `eDeleted`   | 4    | 任务已被删除（调用了 vTaskDelete()）               |
| `eInvalid`   | 5    | 无效状态（任务句柄错误或任务不存在）               |
