# RS485

RS485的电气特性：

- 逻辑 “ 1 ” 以两线间的电压差为+(2~6)V表示；

- 逻辑 “ 0 ” 以两线间的电压差为-(2~6)V表示。

接口信号电平比RS232降低了，不易损坏接口电路的芯片，且该电平与TTL电平兼容，可方便与TTL电路连接。



这是工业级标准。

特点：

-   差分信号（A / B 两根线）
-   抗干扰极强
-   可达1200米
-   支持多机总线（最多32节点）

常见转换芯片：

-   MAX485

# 一、RS485 是什么接口？

RS485 是：

>   一种差分串行通信物理层标准

标准名称来自：

-   Electronic Industries Alliance
-   标准号 EIA-485

它只定义：

-   电气电压
-   差分方式
-   总线结构

它 **不定义协议格式**。

------

# 二、RS485 的核心接口定义

## 1️⃣ 最基本：两根信号线

| 引脚名 | 说明     |
| ------ | -------- |
| A      | 差分线 A |
| B      | 差分线 B |

有时标记为：

-   D+ / D-
-   485+ / 485-
-   Y / Z

⚠ 不同厂家 A/B 可能标反，这是常见坑。

------

## 2️⃣ 电压定义（关键）

RS485 是差分判断：

```
Vdiff = VA - VB
```

| 状态  | 条件    | 表示 |
| ----- | ------- | ---- |
| 逻辑1 | VA > VB | 1    |
| 逻辑0 | VA < VB | 0    |

它看的是“差值”，不是对地电压。

这就是抗干扰强的原因。

------

# 三、RS485 常见芯片接口

以常见芯片为例：

-   MAX485

它一般有：

| 引脚 | 功能                 |
| ---- | -------------------- |
| RO   | 接收输出（接MCU RX） |
| DI   | 发送输入（接MCU TX） |
| DE   | 发送使能             |
| RE   | 接收使能             |
| A    | 差分线 A             |
| B    | 差分线 B             |

------

# 四、控制引脚（半双工关键）

RS485 常见是 **半双工**。

也就是说：

-   发送和接收共用 A/B 两根线
-   不能同时发

所以需要控制：

| 引脚 | 功能                   |
| ---- | ---------------------- |
| DE   | 1 = 发送，0 = 关闭发送 |
| RE   | 0 = 接收，1 = 关闭接收 |

常见接法：

```id="u98s1k"
DE 和 RE 绑在一起
```

用一个 GPIO 控制收发方向。

------

# 五、RS485 接线结构

## 1️⃣ 总线结构（不是星型）

```
设备1 ─── 设备2 ─── 设备3 ─── 设备4
```

不能这样：

```
       ├─ 设备2
设备1 ─┼─ 设备3
       └─ 设备4
```

星型会严重反射。

------

## 2️⃣ 终端电阻

在总线两端必须加：

```
120Ω 终端电阻
```

作用：

-   阻抗匹配
-   减少反射

------

## 3️⃣ 偏置电阻（可选）

用于：

-   空闲时确定默认电平
-   防止悬空

------

# 六、接口形态

RS485 没有固定接口形状。

常见：

-   接线端子
-   RJ45
-   DB9（工业设备）

DB9 在某些设备上用于 RS485，但定义不是统一的。

------

# 七、完整接口层级结构

```
应用协议（Modbus等）
        ↓
UART
        ↓
RS485 收发芯片
        ↓
A / B 差分线
        ↓
双绞线电缆
```

------

# 八、为什么 RS485 可以多机？

因为：

-   输出是三态
-   可以高阻态
-   总线结构
-   差分抗干扰

最多：

>   32 个标准节点（传统规格）

------

# 九、和你现在开发的关系

你如果用 GD32 或 ESP32：

-   UART 外设
-   接 MAX485
-   一个 GPIO 控制 DE
-   就可以做 Modbus RTU

这在工业项目里非常常见。

------

# 十、工程经验提醒（重要）

1️⃣ A/B 不同厂家可能标反
2️⃣ 一定要接地（共地）
3️⃣ 总线必须线型拓扑
4️⃣ 终端电阻只在两端加
