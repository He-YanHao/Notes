# USB接口标准

## USB Type-A

就是最常见的USB接口。

共有四个引脚，其母头引脚引出为：

| 引脚名称 | 作用   | 导线颜色 |
| -------- | ------ | -------- |
| VCC/5V   | 正极   | 红色     |
| D-       | 数据负 | 白色     |
| D+       | 数据正 | 绿色     |
| GND      | 接地   | 黑色     |

支持 USB 3.0 协议的 USB Type-A 接口包含一个 9 线的版本，用于实现USB 3.0的高速数据传输，与USB 2.0保持兼容，但并不包含ID引脚。

| 符号       | 符号名称               | 功能     |
| ---------- | ---------------------- | -------- |
| VBUS       | 电源                   | 提供电源 |
| D-         | 数据传输端负           | 传输数据 |
| D+         | 数据传输端正           | 传输数据 |
| GND        | 地线                   |          |
| STDA_SSRX- | 超高速接收机差分对RX-  | 传输数据 |
| STDA_SSRX+ | 超高速接收机差分对RX+  | 传输数据 |
| GND_DRAIN  | 信号地                 |          |
| STDA_SSTX- | 超高速发射机差分对 TX- | 传输数据 |
| STDA_SSTX+ | 超高速发射机差分对 TX+ | 传输数据 |



## USB Type-B

使用较少，接口为方形，分为 4 线和 5 线版本。



## Mini USB

USB Type-A和USB Type-B的缩小化。



## Micro USB

安卓手机过去常用接口。

新增了 ID 信号线，用于主从识别。

| 引脚名称           | 作用                                           |
| ------------------ | ---------------------------------------------- |
| VCC/5V             | 正极                                           |
| D- / -3.3V         | 数据负                                         |
| D+ / +3.3V         | 数据正                                         |
| ID（仅限 USB OTG） | A型：与地相连；B型：不接地(空)；用来判断主从； |
| GND                | 接地                                           |

### ID引脚

**在标准 Micro-A 线缆中**：ID 引脚通常在设备端是**悬空（NC）** 或连接到 GND。这告诉设备：“你是一个外设，对方是主机。”

**在 USB On-The-Go (OTG) 线缆中**：ID 引脚被**短路到 GND**。当手机通过这种线缆连接U盘等外设时，手机检测到ID脚为低电平，就会切换角色，**自己充当主机**。
