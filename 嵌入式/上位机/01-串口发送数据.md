# 串口发送数据



```python
import serial

# 1. 创建串口对象 (根据你的设备修改参数)
ser = serial.Serial(
    port='COM3',       # 串口号
    baudrate=115200,   # 波特率
    bytesize=8,        # 数据位
    parity='N',        # 校验位 (N无校验/E偶校验/O奇校验)
    stopbits=1,        # 停止位
    timeout=1          # 超时时间(秒)
)

ser.write([0x01, 0x02, 0x03])
#发送二进制数据 

# 3. 确保数据发送完成 (重要!)
ser.flush()  # 等待所有数据写入硬件

# 4. 关闭串口 (完成后)
ser.close()
```
