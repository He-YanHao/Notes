# 中断服务程序

**ISR** 是 **Interrupt Service Routine** 的缩写，中文通常翻译为 **中断服务程序** 或 **中断服务例程**。



**中断 (Interrupt)**： 当硬件设备（如定时器、串口、按键、传感器等）发生需要 CPU 立即处理的事件（例如数据到达、计时完成、状态改变）时，会向 CPU 发送一个**中断信号**。这会强制 CPU **暂停**当前正在执行的任务（或程序流），转而处理这个紧急事件。

**服务 (Service)**： CPU 需要对这个中断事件做出响应和处理。

**例程/程序 (Routine)**： 处理该中断事件的**具体代码**，即一个函数或一段程序。
