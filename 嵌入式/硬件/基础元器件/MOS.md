# MOS

## 基础

MOS管符号可近似看作上下左三端的结构。

```
	D
	|
G - R
	|
	S
```

## NMOS

NMOS可近似看作一个由电压控制的电阻，有三个关键参数：

- 电压：指 G S 两端的电压差。
- 电阻：指 D S 两端的电阻。

- Vgsth

  当电压值大于Vgsth时，电阻接近为零。

  当电压值小于Vgsth时，电阻近似无穷大。

  暂时不考虑等于。

- Rdson

  指MOS管被打开(电阻接近为0时)后 D S 之间的电阻值。

- Cgs：G和S之间的寄生电容，与Rdson大小成反比。由于G端需要先给这个寄生电容充电，所以Cgs越大失真越严重。

# 三极管

## 基础

三极管的符号类似下面：

```
		   12V
			|
			C
			|
控制端 - B --|	
			|
			E
			|
		   GND
```

和MOS管不同的地方在于，三极管是根据电流进行控制的原件，MOS管是根据电压控制的原件。
