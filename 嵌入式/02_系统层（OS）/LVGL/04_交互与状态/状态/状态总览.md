# 状态总览

## 一、什么是状态

状态描述对象当前所处的交互情形，可同时存在多个组合状态。LVGL使用位标志设计，每个状态占一个独立的二进制位。

## 二、完整状态列表

### 基础状态（低8位）

| 状态                 | 值     | 说明                             |
| :------------------- | :----- | :------------------------------- |
| `LV_STATE_DEFAULT`   | 0x0000 | 默认状态，控件的初始基础样式     |
| `LV_STATE_CHECKED`   | 0x0001 | 选中状态，如复选框、开关被勾选   |
| `LV_STATE_FOCUSED`   | 0x0002 | 焦点状态，通过触摸或鼠标点击获得 |
| `LV_STATE_FOCUS_KEY` | 0x0004 | 键盘焦点状态，通过键盘导航获得   |
| `LV_STATE_EDITED`    | 0x0008 | 编辑状态，如文本输入框正在编辑   |
| `LV_STATE_HOVERED`   | 0x0010 | 悬停状态，鼠标指针悬停在控件上   |
| `LV_STATE_PRESSED`   | 0x0020 | 按压状态，控件正在被按下未释放   |
| `LV_STATE_SCROLLED`  | 0x0040 | 滚动状态，控件内容正在滚动       |
| `LV_STATE_DISABLED`  | 0x0080 | 禁用状态，控件不可交互且外观灰化 |

### 用户自定义状态（高4位）

| 状态              | 值     | 说明            |
| :---------------- | :----- | :-------------- |
| `LV_STATE_USER_1` | 0x1000 | 用户自定义状态1 |
| `LV_STATE_USER_2` | 0x2000 | 用户自定义状态2 |
| `LV_STATE_USER_3` | 0x4000 | 用户自定义状态3 |
| `LV_STATE_USER_4` | 0x8000 | 用户自定义状态4 |

### 特殊值

| 状态           | 值     | 说明                 |
| :------------- | :----- | :------------------- |
| `LV_STATE_ANY` | 0xFFFF | 匹配所有状态的通配符 |

## 三、状态分类

| 分类     | 包含状态                                     |
| :------- | :------------------------------------------- |
| 交互状态 | PRESSED、HOVERED、FOCUSED、FOCUS_KEY、EDITED |
| 状态状态 | DEFAULT、CHECKED、DISABLED、SCROLLED         |

## 四、设计特点

| 特点       | 说明                                    |
| :--------- | :-------------------------------------- |
| 位标志设计 | 每个状态占独立二进制位，可通过 `|` 组合 |
| 组合示例   | `LV_STATE_DISABLED | LV_STATE_CHECKED`  |
| 分层结构   | 低8位系统预定义，高4位用户自定义        |
| 预留区间   | 0x0080 至 0x0FFF 预留未使用             |

## 五、状态组合

c

```
// 选中状态下被按下的样式
lv_obj_add_style(btn, &style_checked_pressed, LV_STATE_CHECKED | LV_STATE_PRESSED);
```



| 组合                 | 含义       |
| :------------------- | :--------- |
| `CHECKED | PRESSED`  | 选中且按下 |
| `FOCUSED | CHECKED`  | 焦点且选中 |
| `DISABLED | CHECKED` | 禁用且选中 |
