# 公共宏定义文件说明

## 文件位置

`components/common/include/common_defs.h`



## 设计目的

集中管理项目中所有公共宏定义（任务优先级、内存阈值、时间常量等），避免在各组件中分散硬编码，便于统一维护和全局调整。



## 当前包含内容

| 类别           | 宏定义            | 数值  | 说明                                    |
| -------------- | ----------------- | :---: | --------------------------------------- |
| **任务优先级** |                   |       |                                         |
|                | `PRIO_IDLE`       |   0   | FreeRTOS 空闲任务                       |
|                | `PRIO_HEARTBEAT`  |   1   | 心跳/日志/统计任务                      |
|                | `PRIO_BACKGROUND` |   2   | 后台业务（NTP、传感器）                 |
|                | `PRIO_NETWORK`    |   4   | 网络服务（WiFi、OTA）                   |
|                | `PRIO_LVGL`       |   5   | LVGL 图形渲染                           |
|                | `PRIO_TOUCH`      |   7   | 触摸中断处理                            |
| **系统保留**   |                   | 22-24 | `esp_timer`、`ipc0/1`，**禁止应用占用** |



## 使用方式

### 组件依赖

在需要使用的组件 `CMakeLists.txt` 中添加：

```cmake
REQUIRES common
```

### 代码引用

```c
#include "common_defs.h"

// 使用宏
xTaskCreate(my_task, "my_task", 4096, NULL, PRIO_NETWORK, NULL);
```
