### 与你的 12 个功能直接对应的控件

| 你的功能            | 推荐使用的控件                                               |
| :------------------ | :----------------------------------------------------------- |
| **1. 显示时间**     | `Label`（数字式）、`Meter` + `Image`（指针式）               |
| **2. 更改时间/NTP** | `Roller`（选时）、`Button`（一键对时）、`MessageBox`（提示结果） |
| **3. 更换壁纸**     | `Image`（背景图）、`TileView`（滑动切换预览）                |
| **4. 倒计时**       | `Arc` / `Bar`（进度）、`Spinbox`（设时长）、`Button`（开始/暂停） |
| **5. 亮度调节**     | `Slider`（滑动条）、`Bar`（实时亮度预览）                    |
| **6. 系统状态监控** | `Chart`（内存/CPU曲线）、`Label`（显示数值）、`Bar`（占用率条） |
| **7. 关于本机**     | `Label`（信息文本）、`QRCode`（分享设备信息）                |
| **8. 使用信息统计** | `Label`（统计数值）、`Chart`（亮屏时长趋势）                 |
| **9. 万年历查询**   | `Calendar`（核心控件）、`Label`（显示农历/节日）             |
| **10. Wi-Fi 配网**  | `Keyboard`（输密码）、`List`（显示扫描到的网络）、`Spinner`（连接中） |
| **11. 天气**        | `Image`（天气图标）、`Label`（温度/描述）、`Chart`（未来几天气温曲线） |
| **12. OTA 配置**    | `Bar`（下载进度）、`MessageBox`（升级确认）、`Spinner`（等待） |
| **公共：主菜单**    | `Button` 矩阵或 `Container` + `Button`（九宫格布局）         |

### 💡 使用建议

- **EEZ Studio 中直接拖拽**：在 EEZ Studio 的界面设计器里，你可以直接从右侧控件面板把这些控件拖入屏幕，可视化调整位置和属性，它会自动生成你看到的 `screens.c` 中的代码。
- **样式自定义**：所有控件的外观（颜色、圆角、边框、阴影）都可以通过 LVGL 的样式系统（`Styles`）在 EEZ Studio 中直接配置，无需手写代码。
- **事件绑定**：控件的交互行为（如点击、滑动）需要在 `actions.h` 中声明回调函数，并在 EEZ Studio 中将控件的事件与函数名关联。
