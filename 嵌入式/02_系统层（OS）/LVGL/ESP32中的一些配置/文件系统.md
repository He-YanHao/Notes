# 文件系统

## 要点描述

- LittleFS文件系统适合作为NOR Flash的文件系统。
- FATFS文件系统适合作为SD卡的文件系统。
- ESP32具有VFS兼容层，可以分别注册。



## LittleFS_data

用 `mklittlefs` 工具将 `LittleFS_data` 文件夹直接打包为 littleFS 镜像，保持原目录结构。

GitHub上下载`mklittlefs` 工具，使用：

```powershell
.\mklittlefs.exe -c "D:\HMI\esp32\lvgl\01_circle\LittleFS_data" -b 4096 -p 256 -s 0x900000 littlefs_data.bin
```

打包。

### 核心用法

基本命令格式如下：

```powershell
.\mklittlefs.exe -c <源文件夹> -b 4096 -p 256 -s <镜像大小> <输出镜像名.bin>
```

- `-c`：指定要打包的文件夹路径
- `-b`：块大小，默认4096
- `-p`：页面大小，默认256
- `-s`：输出镜像的最大容量，不能小于源文件夹的实际大小，创建的是固定大小的镜像，剩余空间会用 littleFS 的空闲块填充。
- 最后一个参数是输出的 `.bin` 文件名

使用

```powershell
esptool.py --chip esp32s3 --port COM7 --baud 921600 write_flash 0x620000 littlefs_data.bin
```
