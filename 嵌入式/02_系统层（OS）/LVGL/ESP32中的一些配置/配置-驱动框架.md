## 添加和配置驱动框架

使用

```
idf.py add-dependency "lvgl/lvgl==8.4.0"
idf.py add-dependency "espressif/esp_lvgl_port==2.7.2"
idf.py add-dependency "espressif/esp_lcd_ili9341==2.0.2"
idf.py add-dependency "espressif/esp_lcd_touch_ft5x06==1.1.0~1"
```

添加框架。

继续使用：

```
idf.py menuconfig
```

```
Component config  --->
	LVGL configuration  --->
		Color settings  --->
			[*] Swap the 2 bytes of RGB565 color. Useful if the display has an 8-bit interface (e.g. SPI).
		Demos  --->
			[*] Music player demo
				启用这一个就够了
```

交换RGB字节顺序和启动音乐播放器案例。

接下来只需要依赖 `lvgl` 组件和 `\#include "lv_demos.h"` 头文件即可使用 `lv_demo_music();` 启动音乐案例。

并配置音乐播放器所需的12和16号字体。
