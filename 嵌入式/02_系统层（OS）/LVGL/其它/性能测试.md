# 基于ESP32-S3的LVGL动画性能测试记录

## 测试环境

- 芯片：ESP32-S3
- 图形库：LVGL
- 图片格式：RGB565
- 动画数量：2个（同时播放）
- 单帧尺寸：236 × 100 像素
- 每动画帧数：49帧
- 播放时长：2秒
- 显示接口：SPI
- 传输方式：DMA（`trans_queue_depth = 20`）



## 测试数据

### 全缓冲与分块渲染

均在PSRAM

| 配置                          | 帧率   | CPU    |
| ----------------------------- | ------ | ------ |
| 全缓冲渲染                    | 26 FPS | 89-99% |
| 分块渲染（每次40行，共320行） | 20 FPS | 100%   |

### PSRAM与DRAM

均使用分块渲染，每次40行，共320行

| 存储位置 | 帧率   | CPU     |
| -------- | ------ | ------- |
| PSRAM    | 20 FPS | 100%    |
| DRAM     | 22 FPS | 99-100% |

### rodata放到PSRAM

全缓冲，在PSRAM，text不放到PSRAM

| 配置   | 帧率      | CPU    |
| ------ | --------- | ------ |
| 不开启 | 26 FPS    | 89-99% |
| 开启   | 29-31 FPS | 81-88% |

### text放到PSRAM

全缓冲，在PSRAM，rodata不放到PSRAM

| 配置   | 帧率      | CPU    |
| ------ | --------- | ------ |
| 不开启 | 26 FPS    | 89-99% |
| 开启   | 33-35 FPS | 74-78% |

### rodata和text放到PSRAM

全缓冲，在PSRAM

| 配置     | 帧率      | CPU    |
| -------- | --------- | ------ |
| 均不开启 | 26 FPS    | 89-99% |
| 均开启   | 40-42 FPS | 64-72% |

### -Os 改为 -O2

全缓冲，在PSRAM，rodata和text放到PSRAM

| 配置 | 帧率      | CPU    |
| ---- | --------- | ------ |
| -Os  | 40-42 FPS | 64-72% |
| -O2  | 41-44 FPS | 62-71% |

## 总结

| 指标     | 初始值   | 最终值    | 改善  |
| -------- | -------- | --------- | ----- |
| 帧率     | 26 FPS   | 41-44 FPS | +65%  |
| CPU 占用 | 89-99%   | 62-71%    | -28%  |
| 镜像大小 | 2,804 KB | 2,800 KB  | -4 KB |
