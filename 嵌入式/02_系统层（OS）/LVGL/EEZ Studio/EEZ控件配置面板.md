# EEZ控件配置面板

## btnmatrix

| 标志                        | 功能说明                                                     |
| :-------------------------- | :----------------------------------------------------------- |
| **`HIDDEN`**                | **隐藏按钮**。按钮会不可见且无法交互，但它所占的布局空间会被保留，不会影响其他按钮的排列。 |
| **`NO_REPEAT`**             | **禁用长按重复触发**。当用户长按该按钮时，系统只会触发一次事件，不会持续发送重复事件。 |
| **`DISABLED`**              | **禁用按钮**。按钮会变为不可用状态（样式通常会变灰），无法响应点击或其他交互事件。 |
| **`CHECKABLE`**             | **启用切换（Toggle）功能**。让这个按钮可以像开关一样，被点击后在“选中”和“未选中”两种状态间切换。 |
| **`CHECKED`**               | **设置为“选中”状态**。与 `CHECKABLE` 搭配使用，让按钮在创建时就处于“已选中”的状态。 |
| **`CLICK_TRIG`**            | **改变事件触发时机**。默认情况下，点击事件（`LV_EVENT_VALUE_CHANGED`）在按钮被按下时触发。启用此标志后，会在按钮**被释放**（即完成一次完整的点击）时才触发。 |
| **`POPOVER`**               | **显示弹出提示**。当用户按下按钮时，会在按钮上方弹出一个放大显示文本的浮动气泡，松开即消失，常用于键盘，方便用户确认输入。 |
| **`RECOLOR`**               | **启用文本重新着色**。允许你在按钮的文本中使用 `#RRGGBB 文字 #` 这样的格式，为部分文字单独设置颜色。 |
| **`CUSTOM_1` / `CUSTOM_2`** | **用户自定义标志**。这两个标志位是LVGL保留给开发者自由使用的，你可以用它们来标记和管理按钮的自定义状态，LVGL本身不会操作它们 |



## roller

| 部件              | 官方名称           | 对应元素                 | 说明                                         |
| :---------------- | :----------------- | :----------------------- | :------------------------------------------- |
| **Main part**     | `LV_PART_MAIN`     | 整个滚轮的背景容器       | 滚轮的背景底板，包含背景色、边框、圆角等     |
| **Selected part** | `LV_PART_SELECTED` | **当前选中的那一个选项** | 高亮显示的那个选项的样式（文字颜色、背景等） |



## meter

| 原文                  | 中文翻译       | 说明                                     |
| :-------------------- | :------------- | :--------------------------------------- |
| Name                  | 名称           | 该刻度尺的标识名称                       |
| Scale min             | 刻度最小值     | 量程的起始值                             |
| Scale max             | 刻度最大值     | 量程的结束值                             |
| Scale angle range     | 刻度角度范围   | 刻度弧线覆盖的角度范围                   |
| Scale rotation        | 刻度旋转角度   | 整个刻度尺的起始偏移角度                 |
| Minor tick count      | 小刻度线数量   | 刻度尺上小刻度线的总数                   |
| Minor tick line width | 小刻度线宽度   | 每个小刻度线的像素宽度                   |
| Minor tick length     | 小刻度线长度   | 每个小刻度线的像素长度                   |
| Minor tick color      | 小刻度线颜色   | 小刻度线的颜色                           |
| Major tick distance   | 大刻度线间距   | 每隔多少个小刻度出现一条大刻度（主刻度） |
| Major tick line width | 大刻度线宽度   | 每个大刻度线的像素宽度                   |
| Major tick length     | 大刻度线长度   | 每个大刻度线的像素长度                   |
| Major tick color      | 大刻度线颜色   | 大刻度线的颜色                           |
| Major Tick label gap  | 大刻度标签间距 | 大刻度数值标签与刻度线之间的间距         |

meter的四个组成部分

| 部分               | 对应的表盘元素   | 说明                                                         |
| :----------------- | :--------------- | :----------------------------------------------------------- |
| **Main part**      | 仪表盘背景       | 表盘的"底板"，可以设置背景色、边框、圆角等。                 |
| **Ticks part**     | 刻度线和刻度标签 | 所有刻度线（包括大小刻度）以及大刻度旁边的数字标签。         |
| **Indicator part** | 指针和中心转轴   | 你代码中用 `lv_meter_add_needle_line` 创建的三根指针就属于这个部分，中心的小圆点也在这里设置。 |
| **Items part**     | 弧形指示器       | 表盘上可能绘制的弧线（Arc），比如你之前问过的汽车仪表盘上的彩色能量环 |
