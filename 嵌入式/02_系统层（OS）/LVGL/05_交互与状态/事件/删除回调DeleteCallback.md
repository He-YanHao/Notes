# 删除回调 Delete Callback

删除回调是一种在小部件被删除时自动执行清理工作的机制，无需手动管理事件处理程序。

## 概述

删除回调基于 `LV_EVENT_DELETE` 事件实现，是一个便捷工具。当小部件被删除时，注册的回调函数会自动调用，用于释放与小部件生命周期绑定的资源。

## 使用方法

使用 `lv_obj_add_delete_cb` 为小部件注册删除回调函数。该函数返回指向已创建描述符的指针，如果需要，可以通过 `lv_obj_remove_delete_cb` 删除回调。

多个删除回调可以附加到同一个小部件，当小部件被删除时，所有注册的回调都会被调用。

## 示例

### 删除控件时自动删除定时器

当控件被删除时，其关联的定时器也会被自动删除，无需在删除控件前手动处理定时器。

### 删除控件时释放堆分配的缓冲区

当控件被删除时，动态分配的内存会自动释放，防止内存泄漏。
