# 标志 Flags

标志是对象的二进制属性开关，控制对象是否可点击、可滚动、是否隐藏等底层行为，不涉及具体外观样式。

## 可见与交互

| 标志            | 说明                                   |
| --------------- | -------------------------------------- |
| Hidden          | 隐藏，对象不可见且不响应触摸           |
| Clickable       | 可点击，使非交互控件能够接收输入事件   |
| Click focusable | 控件可通过点击获得焦点                 |
| Checkable       | 控件可被选中（如开关、复选框状态）     |
| Press lock      | 按下锁定，防止按下时被其他控件抢占事件 |

## 滚动相关

| 标志              | 说明                                                         |
| ----------------- | ------------------------------------------------------------ |
| Scrollable        | 可滚动，启用内容滚动                                         |
| Scroll elastic    | 滚动弹性，允许超出边界后回弹                                 |
| Scroll momentum   | 滚动动量，手指离开后继续惯性滚动                             |
| Scroll one        | 单次滚动，一次只滚动一个子对象的高度或宽度，子项需设置 Snappable |
| Scroll chain hor  | 水平滚动链，允许水平滚动传递到父对象                         |
| Scroll chain ver  | 垂直滚动链，允许垂直滚动传递到父对象                         |
| Scroll on focus   | 聚焦时滚动，获取焦点时自动滚动到可见区域                     |
| Scroll with arrow | 使用箭头键滚动                                               |
| Snappable         | 可吸附，设置给子项，让其父容器滚动时能吸附到该子项上         |

## 事件传递

| 标志           | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| Event bubble   | 事件冒泡，未处理的事件向上传递给父对象                       |
| Gesture bubble | 手势冒泡，未处理的手势向上传递给父对象。勾选时手势会向父对象传递，取消勾选时手势被子控件拦截 |

## 布局与绘制

| 标志             | 说明                                                 |
| ---------------- | ---------------------------------------------------- |
| Ignore layout    | 忽略布局，控件不受父布局影响                         |
| Floating         | 浮动，控件不随父对象滚动，类似 CSS 的 position fixed |
| Overflow visible | 溢出可见，子对象超出自身范围仍显示                   |
| Adv hittest      | 高级命中测试，用于圆角、非矩形形状的精确触摸检测     |

## Scrollbar mode（滚动条模式）

控制滚动条的显示方式。

OFF 表示隐藏，永不显示滚动条。ON 表示始终可见，一直显示滚动条。AUTO 表示自动，内容超出时显示滚动条。ACTIVE 表示滚动时显示，闲置后自动隐藏。

## Scroll direction（滚动方向）

控制控件允许的滚动方向。

NONE 表示无滚动。TOP 表示仅向上滚动。LEFT 表示仅向左滚动。BOTTOM 表示仅向下滚动。RIGHT 表示仅向右滚动。HOR 表示水平方向滚动。VER 表示垂直方向滚动。ALL 表示所有方向滚动。

## Scroll snap X（水平吸附对齐）

控制水平滚动时子对象的吸附方式。

NONE 表示禁用吸附。START 表示吸附到子对象的起始边缘。END 表示吸附到子对象的结束边缘。CENTER 表示吸附到子对象的中心。

## Scroll snap Y（垂直吸附对齐）

控制垂直滚动时子对象的吸附方式。

NONE 表示禁用吸附。START 表示吸附到子对象的起始边缘。END 表示吸附到子对象的结束边缘。CENTER 表示吸附到子对象的中心。
