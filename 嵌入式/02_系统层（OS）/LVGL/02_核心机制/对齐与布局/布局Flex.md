# 布局 Flex

Flex 是一种一维布局，可将子元素排列成行或列，类似于 CSS Flexbox。适用于工具栏、列表以及任何由一个轴控制元素位置的堆叠布局。

## 排列方向

flex_flow 属性决定子元素的排列方向和是否换行。

row 表示水平排列，不换行。column 表示垂直排列，不换行。row_wrap 表示水平排列，自动换行。column_wrap 表示垂直排列，自动换行。row_reverse 表示水平反向排列，列顺序反转。column_reverse 表示垂直反向排列，行顺序反转。

## 对齐方式

flex_align 属性控制子元素在主轴和交叉轴上的对齐。

主轴对齐决定子元素在主轴方向上的分布。START 表示子元素靠起始端排列。CENTER 表示子元素居中排列。END 表示子元素靠末端排列。SPACE_BETWEEN 表示两端对齐，中间间距相等。SPACE_AROUND 表示每个子元素两侧间距相等。SPACE_EVENLY 表示所有间距完全相等。

交叉轴对齐决定子元素在交叉轴方向上的对齐。START 表示靠交叉轴起始端对齐。CENTER 表示居中对齐。END 表示靠末端对齐。STRETCH 表示拉伸填满整个交叉轴。

## 生长权重

flex_grow 属性决定当子元素尺寸之和小于容器时，剩余空间的分配方式。设置了 flex_grow 的子元素会按权重比例分到额外的空间。权重越大，分到的额外空间越多。未设置 flex_grow 的子元素不参与分配，保持原始尺寸。

## 内边距与间距

pad_row 控制行与行之间的间距，pad_column 控制列与列之间的间距。

## 忽略布局

设置 ignore_layout 为 true 后，该子元素被 Flex 布局完全忽略。它保留手动设置的坐标，不占用布局槽位，其他子元素会像它不存在一样进行布局。适用于浮动按钮、徽章或绝对定位的装饰元素。

## 强制换行

设置 flex_in_new_track 为 true 后，在换行布局中该子元素会被强制放到新的一行，无论当前行是否还有剩余空间。适用于需要从新行开始的特殊项目。

## 从右到左布局

设置容器的 base_dir 为 RTL 可以反转主轴上的子元素顺序，而不改变代码中子元素的声明顺序。第一个声明的子元素会出现在最右侧。适用于从右到左的语言或镜像布局。

## 工程建议

单行或单列排列优先使用 Flex 而非 Grid。需要等分布局时使用 flex_grow 而非手动计算宽度。浮动元素使用 ignore_layout 脱离布局管理。RTL 布局使用 base_dir 而非手动调整顺序。
