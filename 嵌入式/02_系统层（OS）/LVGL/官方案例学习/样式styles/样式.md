## 三、样式应用方式

### 方式1：全局样式（style_1-10）

```c
static lv_style_t style;
lv_style_init(&style);
lv_style_set_xxx(&style, value);
lv_obj_add_style(obj, &style, 0);
```

### 方式2：本地样式覆盖（style_12）

```c
lv_obj_set_style_bg_color(obj, new_color, 0);  // 覆盖全局样式
```

### 方式3：多样式叠加（style_11）

```c
lv_obj_add_style(obj, &style_base, 0);
lv_obj_add_style(obj, &style_warning, 0);  // 后添加的优先级高
```

### 方式4：部件+状态样式（style_13）

```c
lv_obj_add_style(obj, &style_indic, LV_PART_INDICATOR);  // 指示器部件
lv_obj_add_style(obj, &style_indic_pr, LV_PART_INDICATOR | LV_STATE_PRESSED);  // 按下状态的指示器
```

### 方式5：主题扩展（style_14）

```c
// 继承当前主题，添加自定义样式
lv_theme_set_parent(&th_new, th_act);
lv_theme_set_apply_cb(&th_new, new_theme_apply_cb);
lv_display_set_theme(NULL, &th_new);
```

## 五、模态层实现对比（style_19）

| 方式     | 方法                       | 性能               |
| -------- | -------------------------- | ------------------ |
| recolor  | `lv_obj_set_style_recolor` | 轻量，无额外对象   |
| 顶层背景 | `lv_layer_top()` + bg_opa  | 创建全屏对象，较重 |
