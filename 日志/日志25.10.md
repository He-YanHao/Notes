# 日志25.10

## 10.1

移植FreeRTOS时缺少SysTick中断函数。

解决方法：

在 `stm32xxxx_it.c` 里添加：

```c
#include "FreeRTOS.h"
#include "task.h"

extern void xPortSysTickHandler(void);

void SysTick_Handler(void)
{
    if(xTaskGetSchedulerState() != taskSCHEDULER_NOT_STARTED)
    {
        xPortSysTickHandler();
    }
}
```
