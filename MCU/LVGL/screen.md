# screen

screen

LVGL 在 `lv_init()` 时，会创建了一个默认 screen

Screen 也是一个 lv_obj_t

-   没有父对象
-   尺寸 = 显示分辨率
-   是对象树的根节点







lv_obj_create(obj)

LVGL 里“所有对象本质都是矩形盒子

比如：

| API               | 看起来像 |
| ----------------- | -------- |
| `lv_btn_create`   | 按钮     |
| `lv_label_create` | 文本     |
| `lv_img_create`   | 图片     |
| `lv_arc_create`   | 圆弧     |
| `lv_line_create`  | 线       |
