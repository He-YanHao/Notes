# 🎓 LVGL 完整学习路线（工程师版）

>   目标：
>   **能独立设计、实现、调试一个 LVGL GUI 系统**
>   而不是“能跑 demo”。

我把路线分成 **5 个阶段**，每个阶段都有：

-   学什么
-   做什么
-   到什么程度才算“过关”

------

## 🟢 Phase 0：环境与最小系统（你已完成）

### 学什么

-   LVGL 架构概念（不是 API）
-   Porting 层职责

### 你已经掌握

-   `lv_init()`
-   `lv_port_disp_init()`
-   `lv_timer_handler()`
-   能画一个方块

### 过关标准

>   **我确信：如果屏幕黑了，我知道从哪里查**

✅ **你已通过**

------

## 🟢 Phase 1：对象模型（Object Model）

### 核心思想（非常重要）

>   **LVGL = 树形对象系统**

你要理解：

-   父子关系
-   坐标继承
-   Z-order

### 必学 API（不多）

```c
lv_obj_create
lv_obj_del
lv_obj_set_size
lv_obj_set_pos
lv_obj_align
```

### 必做练习

-   一个 screen
-   上面放 3 个方块
-   改大小 / 改位置
-   删除其中一个

### 过关标准

>   **我不查文档，也知道一个对象为什么在那**

------

## 🟢 Phase 2：Style 系统（决定你是否“写得干净”）

### 学什么

-   Style 是什么
-   Selector（`0`、`LV_PART_MAIN`）
-   局部样式 vs 全局样式

### 必学 API

```c
lv_style_init
lv_style_set_bg_color
lv_style_set_border_width
lv_obj_add_style
```

### 必做练习

-   一个 style
-   作用到多个对象
-   runtime 改 style

### 过关标准

>   **我不会在代码里到处写 set_style_xxx**

------

## 🟢 Phase 3：事件系统（GUI 的“逻辑层”）

### 学什么

-   Event 回调
-   Event bubbling
-   user_data

### 必学 API

```c
lv_obj_add_event_cb
lv_event_get_target
lv_event_get_code
```

### 必做练习

-   点一个方块
-   改颜色
-   再点变回来

### 过关标准

>   **我能不用 printf，也知道事件流怎么走**

------

## 🟡 Phase 4：布局系统（从“摆控件”到“写界面”）

### 学什么

-   Flex（重点）
-   Grid（次要）

### 必学

```c
lv_obj_set_layout
lv_obj_set_flex_flow
lv_obj_set_flex_align
```

### 必做练习

-   一个容器
-   自动排列按钮
-   改分辨率仍然对齐

### 过关标准

>   **我不再手算像素坐标**

------

## 🟡 Phase 5：Image / Label / Font（内容层）

### 学什么

-   Image descriptor
-   内存来源（Flash / RAM）
-   Font 裁剪

### 必学

```c
lv_img_create
lv_img_set_src
lv_label_create
lv_label_set_text
```

### 必做练习

-   显示图片
-   显示中英文
-   动态改内容

### 过关标准

>   **我知道哪些东西吃内存，哪些不吃**

------

## 🟡 Phase 6：Timer / Animation（时间维度）

### 学什么

-   `lv_timer`
-   动画 vs 定时器
-   帧率的真实含义

### 必学

```c
lv_timer_create
lv_anim_init
lv_anim_start
```

### 必做练习

-   方块缓慢移动
-   数字每秒变化

### 过关标准

>   **我能控制“多久刷新一次”**

------

## 🔵 Phase 7：内存模型（嵌入式的生死线）

### 学什么

-   `LV_MEM_SIZE`
-   内部 heap vs 外部 buffer
-   Fragmentation

### 必做练习

-   故意分配大对象
-   观察失败
-   调整配置

### 过关标准

>   **我知道 OOM 为什么发生**

------

## 🔵 Phase 8：文件系统 & 外设接口（可选）

### 学什么

-   lv_fs
-   FatFs 映射
-   流式数据（为摄像头做铺垫）

### 过关标准

>   **我能不用 demo，自行接入一个数据源**
