# 日志

## ESP-IDF 的日志等级

ESP-IDF 一共有 **6 个日志等级**（从低到高）：

| 宏             | 等级    | 说明             | 是否常开   |
| -------------- | ------- | ---------------- | ---------- |
| `ESP_LOGV`     | Verbose | 超详细调试信息   | ❌ 通常关闭 |
| `ESP_LOGD`     | Debug   | 调试信息         | ❌          |
| `ESP_LOGI`     | Info    | 运行状态信息     | ✅          |
| `ESP_LOGW`     | Warn    | 警告（不致命）   | ✅          |
| `ESP_LOGE`     | Error   | 错误（功能异常） | ✅          |
| `ESP_LOG_NONE` | None    | 关闭日志         | —          |
