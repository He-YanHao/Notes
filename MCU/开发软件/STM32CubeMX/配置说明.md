# 配置说明

## SYS

调试接口。

### ST-Link

ST-Link选择 `Serial Wire` 模式，使用：

-   **SWDIO**（数据线）
-   **SWCLK**（时钟线）

JTAG 选择 `JTAG(X pins)`，并连接上对应数目的线。



## RCC

### HSE

-   **Disable**：关闭或禁用。
-   **BYPASS Clock Source**：将来自以有源晶振为主的外部已有的时钟信号直接输入到微控制器的时钟引脚。
-   **Crystal/Ceramic Resonator**：无源晶体，MCU获取时钟信号后经过芯片内部电路处理后作为时钟源。







## GPIO







```

U53 USB_HPY

U54 Key_Number1

U70 Key_Number4
```

| U39     | 通信MCU       |

| U54     | Key_Number1   |
| U55     | Key_Number2   |
| U56     | Key_Number3   |
| U70     | Key_Number4   |
| U1      | Key_Number5   |
