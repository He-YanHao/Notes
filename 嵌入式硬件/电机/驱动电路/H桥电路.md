# H桥电路

## 基础桥型电路

```
      *+1              *+3
      |               |
     /                 \
Q1--+                   +--Q3
     \       /++\      /
      |-----+    +-----
     /       \++/      \
Q2--+                   +--Q4
     \                 /
      |               |
      *-2             *-4
```

- 其中 `*+1` `*+3` 接正极；其中 `*-2` `*-4` 接负极
- Q1~Q4都是三极管，Q1Q2不会一起开，Q3Q4不会一起开，不然会烧毁。
- Q1 Q4开，Q2 Q4关  与   Q2 Q4开，Q1 Q4关电流方向相反。
