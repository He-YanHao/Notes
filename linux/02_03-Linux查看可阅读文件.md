# Linux查看文件

## 查看文件指令

| 指令        | 作用                                                         | 补充                               |
| ----------- | ------------------------------------------------------------ | ---------------------------------- |
| cat 文件名  | 查看文件，一般为文本文件，输出在终端                         | 本质是将文件复制一遍输出           |
| nl文件名    | 查看文件，前面加上行号。                                     |                                    |
| more 文件名 | 以页为单位查看文本文件                                       |                                    |
| less 文件名 | 更强大的文本查看命令                                         | 终端以一种类似记事本的方式输出文本 |
| wc 文件名   | 输出文本的基本属性，包含三个数字，分别是文件包含的 行数 单词数 字节数。 |                                    |
| grep        |                                                              |                                    |
| head/tail   | 仅输出开头或结尾                                             | head -n 行数(单个是数字)  文件名   |
|             |                                                              |                                    |

### less

| 命令                        | 操作           |      |
| --------------------------- | -------------- | ---- |
| 上翻页键(Page Up) 或 b      | 向前一页       |      |
| 下翻页键(Page Down) 或 空格 | 向后一页       |      |
| 上方向键                    | 向前一行       |      |
| 下方向键                    | 向后一行       |      |
| G                           | 移动到末尾     |      |
| 1G 或 g                     | 移动到开头     |      |
| /搜索内容                   | 搜索前文       |      |
| n                           | 重复上一次搜索 |      |
| h                           | 帮助信息       |      |
| q                           | 推出 less      |      |
