# Shell

## 基本

执行shell脚本前需要赋予可执行权限

`chmod +x shell脚本名称`

shell脚本以#!开头，后面跟随解释器的路径。如：

```
#!bin/bash
```

或

```
#!bin/sh
```

> 其中sh是bash的简写，两者性质相同。Linux中也存在其他命令解释器。

shell脚本以#为注释符，与C或C++中的//功能一致。

命令行主要有命令，选项，参数三个基本元素。

## 变量和运算符号

shell脚本中的变量不需要事先声明，可以直接使用。

## 命令

`echo "输出内容"`

echo是标准输出，将内容输出到终端。
