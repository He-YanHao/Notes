```

```

```

```

新建PCB

```
文件-->新的-->PCB
```

用于创建和编辑元器件符号。

```
文件-->新的-->库-->原理图库
```

PCB封装库。

用于创建和编辑元器件封装。

```
文件-->新的-->库-->PCB封装库
```



## 文件说明

- **`PCB_Project.PrjPcb`** - 这是主工程文件，双击它会加载整个工程及其所有相关文件
- **`PCB1.PcbDoc`** - PCB设计文件
- **`Sheet2.SchDoc`** - 原理图文件  
- **`Schlib1.SchLib`** - 原理图库文件
- **`PcbLib1.PcbLib`** - PCB封装库文件
