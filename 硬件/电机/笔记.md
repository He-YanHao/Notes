## 基础

一个这样的电路，会产生磁场：

```
                          -负电极
                          |
                          |
                          |
                          \------------\
               /-----------------------/
               \-----------------------\    ----->  N极
               /-----------------------/
               \-----------------------\
               /-----------------------/
               \-----------------------\
               /-----------------------/    ----->  S极
               \-----------------------\
               /-----------------------/
               \----------\
                          |
                          |
                          +正电极
```

调换正负电极磁场也会跟着改变。
