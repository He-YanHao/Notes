相较于FOC算法而言：

|    **特性**    |         **六步换向**         |            **FOC**             |
| :------------: | :--------------------------: | :----------------------------: |
|  **磁场轨迹**  |      六边形（离散跳跃）      |        圆形（连续旋转）        |
|  **电流波形**  |       方波（120°导通）       |             正弦波             |
|  **转矩脉动**  |     高（15-20%峰值转矩）     |           低（<5%）            |
|    **效率**    |        中（铜损较高）        |      高（电流利用率最优）      |
| **算法复杂度** |      低（查表+定时器）       |   高（Clarke/Park变换+PI环）   |
|  **适用场景**  | 电动工具、风扇、低成本无人机 | 伺服电机、高端无人机、电动汽车 |
